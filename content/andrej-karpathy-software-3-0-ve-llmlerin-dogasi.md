## Ana Fikir

LLM'ler autistic savant gibi: Çok güçlü ama bazı alanlarda çocuksu.

**Clear thinking > prompt engineering** - Ne istediğini bilmek en önemli.

## Software'in Evrimi (Andrej Karpathy)

### Software 1.0 — Klasik Programlama
- İnsanlar kesin talimatlar yazar (C++, Python, JavaScript vb.)
- Kod = program

### Software 2.0 — Sinir Ağı Ağırlıkları
- Sinir ağı ağırlıkları "kod" olur
- Veri + optimizer ile eğitilir (AlexNet, Tesla Autopilot gibi)
- **Hugging Face = 2.0'ın GitHub'ı**

### Software 3.0 (Yeni Paradigma)
- LLM'leri doğal dil (İngilizce) prompt'larla programlıyoruz
- **Prompt = program**
- **En sıcak yeni programlama dili: English**

## LLM'lerin Doğası

LLM'ler **insan ruhlarının stokastik simülasyonu** (autoregressive Transformer + insan metinleri).

### Süper Güçler
- **Rain Man gibi kusursuz hafıza**
- **Ansiklopedik bilgi**

### Zayıflıklar / "Cognitive Quirks"
- **Hallucination** (uydurma)
- **Jagged intelligence** (9.11 > 9.9 gibi basit şeylerde başarısızlık)
- **Anterograde amnesia** (konuşma arası unutkanlık, uzun vadeli öğrenme yok)
- **Prompt injection'a kolay yenik düşme**
